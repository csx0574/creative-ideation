# Method 05 — Lateral Provocations (横向挑衅 / PO 操作)

**Source**: Edward de Bono, 1970, *Lateral Thinking*。https://en.wikipedia.org/wiki/Lateral_thinking

**Originator**: Edward de Bono (马耳他, 1933-2021, 横向思维概念创造者)

## What it is
A method for breaking sequential logic by injecting a *provocative* — a deliberately absurd or wrong intermediate claim — then using the absurdity to find a real path that wouldn't have been visible from the starting point.

## Core mechanism: PO (Provocation Operation)
1. Make a normal statement: "A leads to B"
2. Replace B with something deliberately absurd: "PO: A leads to X" where X is impossible/wrong
3. Don't defend the PO. Use the *movement* from B to X as a probe
4. Ask: "What's actually interesting about moving from B to X?"
5. The path to X often reveals a real but hidden Y

## Common PO patterns
- **Escape**: PO: the problem doesn't exist → what would we do if it didn't?
- **Cause reversal**: PO: the cause is the effect → what would change?
- **Figure reversal**: PO: the figure is the ground → what was overlooked?
- **Cliché reversal**: PO: do the exact opposite of the cliché → what survives?
- **Wishful thinking**: PO: X is already true → what would the next move be?
- **Moment reversal**: PO: the order is reversed → what changes?
- **Strategy reversal**: PO: use the *opposite* of your strategy → what new move appears?

## How to use this method in creative-ideation
1. User says "我想做 X 但不知道怎么开始" or "X 太常规了"
2. Generate 1-3 deliberately absurd POs
3. For each PO, follow the movement, not the PO itself
4. The real idea often appears 2-3 steps into the move, not at the PO

## Best for
- "Too conventional" outputs (great after SCAMPER runs dry)
- Marketing names/copy
- User explicitly asks for 怪 / weird / 反常识
- Breaking out of LLM-slop patterns

## When NOT to use
- User wants convergent/final choice (use Pre-mortem)
- Pure technical engineering (use TRIZ)
- Refining a working draft (use Defamiliarization)

## Example (from de Bono)
- Normal: "How can we improve town traffic flow?"
- PO: "Cars are square" → From square to: lanes that change shape, multi-passenger squares, parking that stacks squares... → Real move: introduce square-shaped carpooling lanes with priority signaling

## Output shape
- **Starting claim**: [the normal logic]
- **PO**: [deliberately absurd]
- **Movement from PO**: [what becomes visible by following it]
- **Salvageable real idea**: [the actual insight that emerged]
