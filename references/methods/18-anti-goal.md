# Method 18 — Anti-Goal (反向目标 / Inversion)

**Source**: 概念可追溯至古罗马 "Inversion" 修辞 (Juvenal); 现代化应用于 Charlie Munger 的 inversion thinking 和 Ray Dalio 的 principles。

**Originator**: 古罗马修辞 → 现代应用 Charlie Munger + Ray Dalio

## What it is
Most people think: "How do I succeed at X?" Munger flipped it: "What would make me *fail* at X? Now avoid that." It sounds simple but most people can't think this way without practice — the brain is biased toward positive planning.

## The move
1. State the goal
2. Invert: "What does failure look like? What guarantees failure?"
3. List all the ways to fail
4. For each failure mode, what's the opposite? Build the opposite behavior
5. Combine: the goal minus all the failure modes = a better-stated goal

## How to use this method in creative-ideation
1. User has a goal: launch a product / finish a book / get fit / learn X
2. Ask: "What are 10 ways to *guarantee* you fail at this?"
3. Each failure mode becomes a thing to actively avoid (often more useful than positive planning)
4. Often reveals: the user was optimizing for the wrong metric

## Best for
- 长期目标 (年度/项目周期)
- 用户在多个目标间选择
- 审视现有策略
- "我之前都失败了" 类的痛点

## When NOT to use
- 创意生成 (这是目标诊断)
- 短平快任务 (用 Crazy 8s)
- 已经很清晰的执行

## Famous application
Munger on investing: "I don't try to figure out who's right. I try to figure out who's wrong. There's only one way to be right, but many ways to be wrong."

## Output shape
- **Goal**: [what you want]
- **Failure mode 1**: [what guarantees failure] → **Avoid by**: [opposite behavior]
- **Failure mode 2**: ...
- **Negative-space goal**: [the goal minus all failure modes]
