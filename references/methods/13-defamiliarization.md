# Method 13 — Defamiliarization (陌生化 / 反常化)

**Source**: Viktor Shklovsky, 1917, 俄国形式主义论文 "Art as Technique" (Искусство как приём)。https://en.wikipedia.org/wiki/Defamiliarization

**Originator**: Viktor Shklovsky (Виктор Шкловский, 1893-1984, 俄国形式主义文学批评家)

## What it is
The purpose of art is to *defamiliarize* the habitual — to make you see the thing you look at every day as if for the first time. Habitualization kills perception; technique restores it. Tolstoy's late stories, which describe familiar things as if seeing them for the first time, are Shklovsky's central example.

## The move
1. Take an object/concept/process the user has been working with
2. Describe it as if you've never seen it before
3. Use absurd precision: physical weight, sounds, what it feels like to touch, what it does *to* you
4. The act of re-description surfaces the assumptions the user was missing
5. Then re-engage with the work — now the hidden assumption is visible

## 5-Dimension Martian Template (v2.1 增强)

原模板只问"动作"。子代理在 Problem 3 vault 案例中证明，5 个维度同时问能抓到更高质量的洞察（"飘向右上角 4.7 秒"就是维度 5 抓到的）。

1. **动作 (Action)**: 主体做什么（手、脚、眼睛）
2. **时间 (Time)**: 每个动作多少秒（浮点精度）。**精度越细，洞察越深** — "0.8 秒" 比 "几秒" 强 10 倍
3. **空间 (Space)**: 注意力指向屏幕的哪里（像素坐标或区域）
4. **能量 (Energy)**: 肌肉 / 瞳孔 / 呼吸 / 心率的变化
5. **沉默 (Silence)**: 不动作时注意力飘向哪里（这是 capture 机会的金矿）— 桌面空白？屏幕边框？天花板？这一维度的**空白区域**就是产品的入口

**如何用 5 维度**:
- 不是 5 个独立描述，是 1 段连贯的"电影剧本"：动作触发时间变化，时间触发空间变化，空间触发能量变化，能量触发沉默（注意力空白）
- 沉默维度必须写出来 — 否则模板退化成"普通动作描述"
- 5 维度填完一般 100-200 字（不是 1 句）

## How to use this method in creative-ideation
1. User says "我的 X 不够好 / 太平庸 / 缺细节"
2. Pick the most over-familiar element of X
3. Describe it as a Martian anthropologist would: "This is a 200g object. When the human operates it, their right hand does X. Their pupils dilate. A small sound is made. They pause. Why?"
4. The detail-by-detail description almost always surfaces something to fix

## Best for
- 打磨阶段 (refinement)
- 写完发现平淡
- 设计 "看起来太普通"
- 用户已经知道大方向但缺细节

## When NOT to use
- 早期 (用户还不知道在做什么)
- 决策类问题 (用 Pre-mortem)
- 时间紧迫

## Famous examples
- Tolstoy's "What is art?" — describes a child's broken pencil as if from the child's perspective
- David Foster Wallace's "A supposedly fun thing I'll never do again" — describes a cruise ship
- Ocean Vuong's poetry — every object is described as if first-encountered

## Output shape
- **Familiar element being defamiliarized**: [the thing]
- **Martian description**: [the absurd-precise re-description]
- **Hidden assumption surfaced**: [what the description makes visible that wasn't before]
- **Design fix**: [what to change in the work]
