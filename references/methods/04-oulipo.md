# Method 04 — OuLiPo (潜在文学工场)

**Source**: OuLiPo = Ouvroir de littérature potentielle, 1960 年由 Raymond Queneau 和 François Le Lionnais 在巴黎创立。https://en.wikipedia.org/wiki/Oulipo

**Originator**: Raymond Queneau (法国作家, 《Zazie dans le métro》作者) + François Le Lionnais (数学家)

## What it is
A literary workshop that imposed *constraints* on writing to liberate creativity. Famous constraint: Georges Perec wrote *La Disparition* (1969) — a 300-page novel that never uses the letter "e" — the most common letter in French.

## Core constraints (selection)
- **E-avoidance**: never use a specific letter
- **Lipogram**: avoid a specific letter
- **Pangrammatic window**: use every letter in a fixed window
- **Snowball**: each word one letter longer than the last
- **Palindrome**: sentence reads the same backwards
- **N+7**: replace every noun with the noun 7 entries later in a dictionary
- **S + 7**: replace every noun with the entry 7 entries before
- **Polyptoton**: repeat a word in multiple grammatical forms
- **Beaumont's law**: limit the number of times a sound appears
- **Homovocalism**: use only words with one vowel
- **Prisoner's constraint**: write a story where characters cannot leave a single room
- **Dialogue-only**: no narration, only spoken lines
- **Reverse**: end is beginning is end

## How to use this method in creative-ideation
1. User says "我想写个 [X] 但写不出来 / 太普通"
2. Pick a constraint that would force the work into unfamiliar shape
3. Apply for one full draft — don't break the rule
4. The rule forces invention; the draft may keep or shed the rule later

## Best for
- Writing (fiction, essays, articles, scripts)
- Naming/branding
- Game design rules
- Any task where "too easy" is the enemy

## When NOT to use
- Time-sensitive deliverables
- Tasks with strict existing constraints (use constraint subtraction to find *new* room)
- Pure technical work (TRIZ is better)

## Famous OuLiPo result
Italo Calvino's *If on a winter's night a traveler* (1979) — uses begin-in-medias-res for every chapter, then interrupts each with a different opening. Without OuLiPo's constraint work, the structure doesn't exist.

## Output shape
- **Target**: [what they're making]
- **Constraint**: [the rule, specific and unbreakable]
- **First draft attempt under constraint**: [the output, even if rough]
- **What the constraint forced into existence**: [the invention that wouldn't have happened without it]
